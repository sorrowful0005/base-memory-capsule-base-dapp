# Base Memory Capsule

## App Name

Base Memory Capsule

## Tagline

Seal it for later

## Description

Seal a message on Base, set a future unlock date, and reopen it later with a clear onchain timestamp.

Character count: 104

## Category

Archive / reflection / onchain utility

## Short Pitch

Base Memory Capsule gives people a calm way to preserve one message for the future and reopen it later with a clear timestamped onchain record.

## Production Environment Variables

- `NEXT_PUBLIC_MEMORY_CAPSULE_CONTRACT_ADDRESS`: deployed memory capsule contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
